Help

Typesetting follows this process:

  1. Assign a typesetter.
  2. Review the uploaded galleys.
  3. If required, assign proofreader.
  4. If required, ask the typesetter to make corrections.
  5. Loop until you're happy with the galleys.

As long as there are galleys assigned, you can complete the typesetting stage at any point.
